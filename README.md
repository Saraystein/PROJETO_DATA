# Conversor de Datas (ISO - BR) e (BR - ISO)

Aplicação Flask que converte datas entre os formatos americano (YYYY-MM-DD) e brasileiro (DD/MM/YYYY) usando Expressões Regulares.

## Estrutura
- `app.py`  
- `static/`
  - `index.html`
  - `style.css`
  - `script.js`
- `requirements.txt`
- `.gitignore`
